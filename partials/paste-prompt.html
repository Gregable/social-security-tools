To use the calculator, you must provide some data from your Social Security record.

If you aren't ready for that yet, select a demo data set at the bottom of the page.

Retrieve your Social Security data:

  1. Login to ssa.gov. You may need to create an account if you haven't already.
  2. Click to navigate to the "Earnings Record" page of the site.
  3. Copy the earnings table. It should look similar to the example below, with different numbers and years. You can select the table by dragging your mouse over the entire table to select, or you can just use 'Control+A' to select the entire page. Either way will work. Copy the text with 'Control+C'.
  4. Return to this page, paste the result into the text area below with 'Control+V'.

🔒 Your data never leaves your own computer: the report is generated entirely by JavaScript in your browser.

Alternatively, just paste any text formatted string with year and earnings in the same line, for example: 

2000 $10,000
2001 $15,000
